Тест Админ Панели

Откройте консоль браузера (F12) и выполните следующие команды для тестирования:

Тестовые команды:

// 1. Проверить доступность API
fetch('http://localhost:3030/api/admin/employees').then(r => r.json()).then(console.log);

// 2. Открыть главную страницу админ-панели
window.open('http://localhost:5555', '_blank');

// 3. После входа в админ-панель с табельным номером "admin12qw", 
//    проверить инициализацию в консоли:
document.querySelectorAll('.menu-item').length
document.querySelectorAll('.content-section').length

// 4. Проверить переключение секций:
window.switchSection('departments')
window.switchSection('positions')

// 5. Проверить функции загрузки данных:
window.loadEmployees()
window.loadDepartments() 
window.loadPositions()

Проверочный лист:

  • ✓ Backend API работает на порту 3030
  • ✓ Frontend работает на порту 5555
  • ⚠️ Админ-панель требует тестирования

Инструкция по тестированию:

  1. Откройте http://localhost:5555
  2. Введите табельный номер: admin12qw
  3. Нажмите "Войти"
  4. Проверьте работу кнопок в левой панели
  5. Проверьте загрузку данных в таблицах
